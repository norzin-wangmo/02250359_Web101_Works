## Project Description

This project is a simplified TikTok-style web interface developed using Next.js and Tailwind CSS. The application demonstrates modern frontend development concepts including component-based architecture, routing, and form validation.

The project replicates basic TikTok web layout features such as:

- Sidebar navigation
- Video feed interface
- User profile page
- Upload page
- Login and Signup forms
- Form validation using React Hook Form

## Technologies Used

- Next.js
- React.js
- Tailwind CSS
- React Hook Form
- JavaScript (JSX)

## Features Implemented

### Navigation System
The application includes a sidebar navigation that allows users to navigate between different sections of the application including:

- For You
- Following
- Explore
- Live
- Upload
- Profile

### Video Feed
A mock video feed is implemented using reusable React components to simulate TikTok's vertical video interface.

### Login and Signup Forms
User authentication forms are implemented using React Hook Form with validation features including:

- Required fields
- Email validation
- Password length validation
- Password confirmation matching

### Upload Interface
Users can simulate uploading a video through a simple file input interface.

## Problems Faced

Several challenges were encountered during development:

1. Understanding the folder structure of Next.js with the App Router.
2. Learning how to properly organize components and pages.
3. Implementing form validation using React Hook Form.
4. Styling components using Tailwind CSS classes.

## Solutions

These challenges were resolved by:

- Following official documentation for Next.js and React.
- Breaking the project into smaller reusable components.
- Testing each feature step by step.
- Using Tailwind CSS utility classes for faster UI styling.

## How to Run the Project

1. Clone the repository in the terminal

git clone <repository-url>

2. Install dependencies

npm install

3. Start the development server

npm run dev

4. Open the application in the browser at `http://localhost:3000`
